Minimum size

the minimum size is a length set to preserve the different species and to avoid overfishing which leads to the depletion of fish populations. if you catch a fish under the minimum size, gently release it back into the sea. if the fish died on its way to land or is too damaged to survive you can keep it.

Different fish have different minimum sizes. the list is large and covers many of the species, so it is important that you check the list before you go fishing, the full list can be found at: http://www.fiskeridirektoratet.no/
